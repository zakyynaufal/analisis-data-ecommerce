# Proyek Analisis Data E-Commerce ✨

## Deskripsi Project
Project ini merupakan submission analisis data menggunakan dataset e-commerce.  
Analisis dilakukan menggunakan Python dan divisualisasikan melalui dashboard Streamlit.

## Setup Environment - Shell/Terminal

```bash
mkdir proyek_analisis_data
cd proyek_analisis_data
pipenv install
pipenv shell
pip install -r requirements.txt
```

## Run Streamlit App

```bash
streamlit run dashboard/dashboard.py
```
